# knight-travails

## 📌 Overview
This project implements a function `knightMoves(start, end)` that finds the **shortest path a knight can take on a chessboard** from one square to another.

It uses the concept of **graphs** and the **Breadth-First Search (BFS)** algorithm to guarantee the shortest path.
